Инструкция по работе с системой учёта ТМЦ

1. Общие сведения

Система учёта ТМЦ (товарно-материальных ценностей) предназначена для инвентаризации оборудования спортивных комплексов компании «Спорт всегда». Состоит из двух частей:

  • Telegram-бот — для полевой работы: добавление, перемещение, списание и поиск оборудования с фотографированием и голосовым вводом.
  • Веб-дашборд (эта страница) — для просмотра, фильтрации и управления данными из браузера.

Все данные хранятся в Google Таблице и доступны в режиме реального времени. Фотографии хранятся на Битрикс24 Диске.

2. Инвентарный номер

Каждая единица оборудования должна иметь уникальный инвентарный номер.

Инвентарный номер должен совпадать с кодом на наклейке (стикере), которая физически клеится на оборудование. Перед внесением в систему убедитесь, что наклейка наклеена.
  • Номера назначаются последовательно: 001, 002, 003 и т.д.
  • Система автоматически предлагает следующий свободный номер.
  • Дубликаты запрещены — система проверяет уникальность.
  • Добавить оборудование без номера нельзя.

3. Категории оборудования

Оборудование классифицируется по двухуровневой системе:

  • Категория — основной тип (Климатическое, Строительное, Мебель и т.д.)
  • Подкатегория — конкретный вид (Шкаф, Перфоратор, Кондиционер и т.д.)

Примеры: Мебель → Шкаф, Климатическое → Кондиционер, Строительное → Перфоратор.

4. Спецификация (шильдик)

При добавлении оборудования рекомендуется указывать данные с заводской таблички (шильдика):

  • Тип ТМЦ — полное наименование типа оборудования (напр. «Кондиционер настенный сплит-система»)
  • Модель — модель и/или серийный номер с шильдика (напр. «Daikin FTXB35C»)

Эти данные помогают точно идентифицировать оборудование при ремонте, заказе запчастей и списании.

5. Добавление оборудования (вкладка «Внести»)

  1. Укажите Фамилию и Имя того, кто проводит операцию
  2. Выберите комплекс, где находится оборудование
  3. Введите инвентарный номер (система подставит следующий свободный)
  4. Выберите категорию и подкатегорию
  5. Заполните Тип ТМЦ и Модель с шильдика (рекомендуется)
  6. Укажите состояние и добавьте описание
  7. Приложите ссылки на фото (наклейка, шильдик, общий вид)
Через Telegram-бот фото загружаются автоматически. В дашборде нужно вставить ссылки на уже загруженные фото из Битрикс24 Диска.

6. Перемещение оборудования (вкладка «Переместить»)

  1. Укажите Фамилию и Имя
  2. Введите инвентарный номер
  3. Выберите откуда и куда перемещается
  4. Укажите причину перемещения
  5. При необходимости добавьте примечание

В Telegram-боте дополнительно проверяется, что оборудование действительно числится на комплексе-источнике.

7. Списание оборудования (вкладка «Списать»)

  1. Укажите Фамилию и Имя того, кто инициирует списание
  2. Введите инвентарный номер
  3. Выберите причину списания (износ, поломка, утеря и т.д.)
  4. Опишите обстоятельства
  5. Приложите ссылку на фото

После списания:

  • Оборудование остаётся в перечне со статусом «Списано»
  • Формируется Акт списания, который можно распечатать или сохранить как PDF
  • Акт содержит все данные и подпись ответственного

8. Просмотр и фильтрация (вкладка «Перечень»)

Таблица со всем оборудованием. Доступные фильтры:

  • Поиск — по инвентарному номеру, описанию, типу ТМЦ или модели
  • Комплекс — фильтр по объекту
  • Категория — тип оборудования
  • Состояние — техническое состояние (включая «Списано»)
  • Ответственный — кто вносил запись

Кнопка ↻ Обновить — перезагрузить данные из таблицы.

9. Telegram-бот — команды

  • /start — добавить новое оборудование (пошаговый ввод с фото)
  • /move — оформить перемещение между комплексами
  • /writeoff — списать оборудование (с фото и актом)
  • /find — найти оборудование по инвентарному номеру
  • /dashboard — открыть этот дашборд
  • /guide — открыть эту инструкцию
  • /cancel — отменить текущую операцию

Бот поддерживает голосовые сообщения — описание можно надиктовать. Фото наклейки распознаётся автоматически (QR-код или OCR).

11. Частые вопросы

  • Что делать, если номер уже занят? — Проверьте, не было ли оборудование уже внесено. Если это другое оборудование, используйте другой номер.
  • Можно ли изменить данные после внесения? — Напрямую из дашборда нельзя. Обратитесь к администратору или отредактируйте Google Таблицу.
  • Где хранятся фото? — На Битрикс24 Диске. В таблицу записываются ссылки на фото.
  • Как найти списанное оборудование? — В фильтре «Состояние» выберите «Списано».
