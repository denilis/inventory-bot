Инструкция по работе с системой учёта ТМЦ

1. Общие сведения

Система учёта ТМЦ состоит из двух частей:

  • Telegram-бот — для полевой работы: добавление, перемещение, списание и поиск оборудования с возможностью фотографирования и голосового ввода.
  • Веб-дашборд (эта страница) — для просмотра, фильтрации и управления данными из браузера.

Все данные хранятся в Google Таблице и доступны в режиме реального времени.

2. Telegram-бот — команды

  • /start — добавить новое оборудование (пошаговый ввод: комплекс, фото, инв. номер, категория, состояние, описание)
  • /move — оформить перемещение оборудования между комплексами
  • /writeoff — списать оборудование (с обязательным фото и актом списания)
  • /find — найти оборудование по инвентарному номеру
  • /dashboard — открыть этот дашборд
  • /guide — открыть эту инструкцию
  • /cancel — отменить текущую операцию

Бот поддерживает голосовые сообщения — описание можно надиктовать.

3. Перечень оборудования

Вкладка Перечень — таблица со всем оборудованием. Фильтры:

  • Поиск — по инвентарному номеру, описанию или комплексу
  • Комплекс — фильтр по объекту
  • Категория — тип оборудования
  • Состояние — техническое состояние (включая «Списано»)
  • Ответственный — кто вносил запись

Кнопка ↻ Обновить — перезагрузить данные из таблицы.

4. Перемещение оборудования

Вкладка Переместить — форма перемещения:

  1. Введите инвентарный номер
  2. Выберите откуда и куда перемещается
  3. Укажите причину перемещения
  4. При необходимости добавьте примечание

В Telegram-боте дополнительно проверяется, что оборудование действительно находится на комплексе-источнике.

5. Списание оборудования

Вкладка Списать — оформление акта списания:

  1. Введите инвентарный номер
  2. Выберите причину списания (износ, поломка, утеря и т.д.)
  3. Опишите обстоятельства
  4. Укажите ФИО ответственного
  5. Приложите ссылку на фото оборудования

После списания оборудование остаётся в перечне со статусом «Списано». Акт можно распечатать.

6. Добавление оборудования

Вкладка Внести — добавление нового оборудования:

  1. Выберите комплекс и категорию
  2. Введите инвентарный номер (проверяется на уникальность)
  3. Укажите состояние и описание
  4. Приложите ссылки на фото (из Битрикс24 Диска)

7. Комплексы

В системе зарегистрированы объекты:

Остров, Маяк, Юрловский, Косино, Арктика, Янтарь
